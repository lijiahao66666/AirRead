## 目标
- 用“漫画（分镜）”完全替换当前插画分析：章节级分镜脚本 + 关键格出图 + 点选补图。
- 交互收敛到 AI 面板（类似问答面板），去掉胶囊入口与自动分析章节。

## 最终体验
- AI 面板新增入口/页签：漫画。
- 面板内一键“分析当前章节”→展示分镜网格→自动出图若干格→用户点选任意格继续出图→可重新分析。

## 核心流程（两阶段）
### 1) 漫画导演：章节分镜脚本
- 输出：JSON 数组 panels（分镜格子），每格含：
  - anchor（段落索引/范围）
  - narrativeRole（铺垫/升级/高潮/收束）
  - subject/action（不新增剧情事实）
  - setting/time/weather（不引入新地点）
  - shot/camera（景别/机位）
  - lighting/mood/composition（导演信息：光影/氛围/构图/速度线）
  - caption（可选：短对白/旁白）
- 规则：严禁新增剧情事实；允许补导演信息以提升漫画感。

### 2) 提示词导演：单格扩写生图 prompt
- 把每格蓝图扩写为高信息密度 prompt（主体/动作/环境/时间天气/镜头/光影/氛围/构图要点齐全），并做自检避免引入新实体。

## 本地模型如何使用
- 本地模型优先用于“漫画导演”（脚本生成）与镜头筛选：文本输出，成本低。
- 对超长章节：分层压缩（分块要素笔记→聚合→生成分镜）适配 4096 上下文。
- prompt 扩写优先线上（更丰富）；离线场景用本地扩写。

## 设置项（只保留 3 个，并写清作用提示）
- 分镜格数（推荐默认 8，选项：6 / 8 / 9）
  - 提示文案：格数越多，叙事更连贯但分析更慢；格数越少，更像“挑重点”。
- 自动出图数（推荐默认 2，选项：0 / 1 / 2）
  - 提示文案：自动出图会直接消耗额度/积分；设置为 0 只生成分镜脚本，点选某格再出图。
- 强制本地分析（开关，默认关闭）
  - 提示文案：开启后分镜脚本与提示词扩写优先用本地模型（更省钱、更隐私），但画面丰富度可能不如线上。
